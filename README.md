# Company Wise Medicine Form Builder

## Overview

Professional medicine management and A4 print generation system.

## Tech Stack

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

### Frontend

- React + Vite
- Material UI
- React Query
- Axios
- React Select

## Features

- Company Management
- Medicine Item Management
- Company Wise Filtering
- Duplicate Prevention
- Professional A4 Print
- Automatic Pagination
- PDF Export through Browser Print

## Folder Structure

Backend
Frontend

## Installation

Backend:

npm install
npm run dev

Frontend:

npm install
npm run dev

## Print System

- A4 Size
- 80 Items Per Page
- 40 Items Left
- 40 Items Right
