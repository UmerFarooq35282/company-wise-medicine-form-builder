# Company Wise Medicine Form Builder & A4 Print Generator

## Overview

A production-ready MERN Stack application to manage company-wise medicine items and generate professional A4 printable medicine forms.

---

## Tech Stack

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose

### Frontend

- React.js (Vite)
- Material UI
- React Query
- React Select
- Axios

---

## Features

- Company Management
- Medicine Management
- Company-wise Filtering
- Professional A4 Preview
- Browser Print Support
- Multi-page Printing
- PDF Download
- Responsive UI
